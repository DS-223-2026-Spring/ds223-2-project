# ETL and orchestration

- [Home](index.md) · [API models](api_models.md) · [Schema diagram](imgs/star_schema.png)

The default repository layout is centered on the Dockerized PostgreSQL, FastAPI, and Streamlit services. Batch ETL and workflow orchestration can be added in a separate package or service by the data and orchestration roles; use `docs/imgs/` and this folder for design notes and diagrams.
