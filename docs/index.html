AdVise

AdVise uses Docker and Docker Compose for PostgreSQL, a one-shot marketing ETL under AdVise/etl (before the API), the FastAPI and Streamlit apps under AdVise/, and optional Jupyter under AdVise/ds.

How it works?

End-to-end tutorial

Services

  • Database — PostgreSQL
  • API — FastAPI connected to PostgreSQL
  • Front — Streamlit UI calling the API (Compose service front)
  • pgAdmin — UI to inspect and manage data

For the full documentation site, run mkdocs serve from the repository root (the built home page comes from index.md). The file index.html in this folder is a standalone copy for opening in a browser without running MkDocs.
